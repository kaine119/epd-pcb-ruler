# EPD PCB ruler
An SUTD/EPD-branded PCB ruler. 

## Features
* 200mm ruler with flats on both ends of the scale
* Metric screw size references
* Common(-ish) SMT component size references
* Breadboard layout through-hole grid for size references (also handy for folding THT component legs)
* Protractor
* Font height references (useful for adding text features on 3D prints, PCBs, etc.)
* Resistor chart

## Ordering guide
* Any PCB vendor (e.g. JLCPCB, PCBWay) will do
* Gerbers available in the Releases section of this repo, otherwise generate your own with KiCad
* ENIG finish, black solder mask for best results
